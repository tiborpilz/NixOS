Core layout

What lives where

  • hosts/ defines machine-specific NixOS systems.
  • modules/ contains reusable NixOS and shared modules.
  • home/ packages the Home Manager side of the environment.
  • packages/ adds custom utilities and package overrides.
  • tests/ keeps upgrade coverage for key services.

Flake model

One flake, multiple targets

The flake exports host configurations, Home Manager profiles, custom packages, development shells, and checks. It also pulls in nix-darwin, deploy-rs, sops-nix, and editor overlays to keep the setup modular.
